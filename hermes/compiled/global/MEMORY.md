# Global Memory Policy

## What Goes Where

- Git compiled markdown: canonical human-readable working truth.
- Raw evidence: Google Drive, ASM, local workspaces, exported logs, PDFs, screenshots.
- Honcho: active user/agent state and peer conclusions.
- Neo4j: relationship graph and claim/source/tool/agent links.
- Redis or other cache: fast, source-linked, freshness-stamped summaries only.

## Promotion Ladder

0. Raw observation
1. Extracted claim
2. Corroborated claim
3. Canonical truth in compiled markdown
4. Operational rule/skill

## Rules
- Do not promote claims without source refs.
- Do not silently resolve contradictions.
- Do not let cache become truth.
- Do not let maintenance agents rewrite critical instructions without approval/evals.
