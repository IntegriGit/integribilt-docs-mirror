# Context Pack Template

Generated context packs should include:

```text
AGENTS.md
PROJECT.md
TASK.md
MEMORY_ACCESS.md
SOURCES.md
KNOWN_GAPS.md
OUTPUT_CONTRACT.md
```

## Rules
- Keep packs narrow to the task.
- Do not include broad legal/financial/personal material unless scoped.
- Prefer source references over copied raw evidence.
- Cloud agents write back artifacts/reviews, not direct canonical edits.
