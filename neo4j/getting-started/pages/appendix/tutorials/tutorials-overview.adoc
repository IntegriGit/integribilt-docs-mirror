== Overview

This tutorial is based on _the Movie Graph_.
You'll find out how to create, query, and delete data in Neo4j.

* xref:cypher/schema.adoc[Defining a schema] shows how to define a schema by using indexes and constraints.

* xref:cypher/updating.adoc[Updating the graph] describes how to update information in the graph by changing, removing, and adding nodes, relationships, and properties. It also addresses how to avoid duplication.
