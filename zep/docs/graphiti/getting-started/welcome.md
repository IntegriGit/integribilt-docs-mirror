# Welcome to Graphiti!

> Graphiti is Zep's open-source framework for temporal knowledge graphs — Context Graphs — for AI agents, with real-time updates and hybrid retrieval.

Want to use Graphiti with AI assistants like Claude Desktop or Cursor? Check out the [Graphiti MCP Server](/graphiti/getting-started/mcp-server).

Graphiti is the open-source framework for building temporal knowledge graphs — Context Graphs — that power Zep. It makes real-time incremental updates to graphs for AI agents, with no batch recomputation, so relationships and facts evolve as your data does.
